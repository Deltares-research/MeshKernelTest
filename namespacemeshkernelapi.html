Mesh2d orthogonalization.

The function modifies the mesh for achieving orthogonality between the edges and the segments connecting the face circumcenters. The amount of orthogonality is traded against the mesh smoothing (in this case the equality of face areas). The parameter to regulate the amount of orthogonalization is contained in meshkernelapi::OrthogonalizationParameters::OrthogonalizationToSmoothingFactor

Parameters
[in]meshKernelIdThe id of the mesh state
[in]projectToLandBoundaryOptionThe option to determine how to snap to land boundaries
[in]orthogonalizationParametersThe structure containing the orthogonalization parameters meshkernelapi::OrthogonalizationParameters
[in]polygonsThe polygon where to perform the orthogonalization
[in]landBoundariesThe land boundaries to account for in the orthogonalization process
Returns
Error code
