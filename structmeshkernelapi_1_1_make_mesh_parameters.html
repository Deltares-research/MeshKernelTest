int meshkernelapi::MakeMeshParameters::grid_type

The type of grid to create.

The possible types are:

  • square = 0
  • wieber = 1
  • hexagonal type 1 = 2
  • hexagonal type 2 = 3
  • triangular = 4

Square is the suggested default.
